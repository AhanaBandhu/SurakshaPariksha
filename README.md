## 🚀 SurakshaPariksha

> **"Where every exam is monitored, secured, and trusted."**

ExamNova is a **Computer-Based Testing (CBT) platform** designed to conduct secure, scalable, and real-time online examinations. It integrates **proctoring, auto-save mechanisms, and instant evaluation** to ensure a seamless and cheat-resistant exam experience.

Built for **controlled environments (LAN/remote)**, ExamNova provides role-based access for administrators, teachers, and students.

---

## 🌟 Key Features

### 🔐 Advanced Proctoring System
- Real-time monitoring during exams  
- Suspicious activity detection  
- Ensures **fair and cheat-resistant environment**

### 💾 Auto-Save Intelligence
- Continuous answer saving  
- Prevents data loss during network interruptions  

### ⚡ Instant Evaluation
- Automatic result generation  
- Immediate feedback for objective questions  

### 👥 Role-Based Access Control
- **Admin** → Manage users, exams, subjects  
- **Teacher** → Create and monitor exams  
- **Student** → Attempt exams securely  

### 🌐 Full-Stack Architecture
- Seamless integration of frontend and backend  
- REST API-based communication  

### 📊 Scalable & Reliable
- Designed for multiple users in real-time  
- Optimized for performance in LAN environments
